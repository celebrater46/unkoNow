【使い方】
Google Chromeでツイッターを開きます。
IMEの日本語入力を有効にしておきます（半角英数でも動きますが、ローマ字の「unkonau」が投稿されます）。
unko.vbsをダブルクリックします。

一度起動すると1時間に1回「うんこなう」とツイッター上でつぶやく動作をします。
タスクマネージャーから強制終了しない限り24時間つぶやき続けます。
特定のタイミングでキー操作を実行する仕組みなので、ブラウザのタブを変更したり、途中でマウスをクリックしたりキー入力したりすると何かおかしなことになります。

強制終了するには Ctrl + Alt + Delete でタスクマネージャーを開き、アプリの項目にある「Microsoft Windows Based Script Host」を終了するか、強引に再起動するか、PCの電源を無理矢理落とすしかありません。


【利用規約】
このスクリプトは一度動き始めると24時間「うんこなう」をつぶやき続ける悪夢のツールです。何かPCがおかしなことになったり、ツイッターのアカウントが凍結されたり、貴殿の名誉を損ねることになったり、その他何か不都合が起きても作者は一切責任を取りません。
このツールを使用した時点で規約に同意したことになります。
